# Asistente de Turnos con IA (Automatización + APIs)

Descripción

Sistema automatizado de gestión de turnos que permite a los usuarios agendar, consultar y cancelar citas mediante interacción conversacional con inteligencia artificial.

Integra servicios externos y almacenamiento de datos para simular un entorno real de operación.

Problema

La gestión manual de turnos genera errores, superposición de horarios y alta carga administrativa.

Solución

Automatización completa del proceso de turnos mediante IA, permitiendo gestionar citas de forma eficiente y sin intervención humana.

Funcionamiento

1. Usuario inicia conversación
2. IA identifica intención (agendar, cancelar, consultar)
3. Se valida disponibilidad en calendario
4. Se registra o modifica el turno
5. Se almacena la información del cliente


Valor de negocio

- Reducción de errores humanos
- Automatización de agenda
- Mejora en experiencia del cliente
- Optimización del tiempo operativo

Tecnologías utilizadas

- n8n
- Google Calendar API
- Google Sheets
- OpenAI / LLMs
- Webhooks


Enfoque

Proyecto orientado a resolver un caso real de gestión de turnos mediante automatización inteligente e integración de sistemas.

Nota

El sistema fue desarrollado utilizando herramientas de IA como soporte para desarrollo, optimización y pruebas.
